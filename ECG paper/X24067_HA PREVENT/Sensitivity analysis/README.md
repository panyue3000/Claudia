# X24067 HA PREVENT — Insurance Sensitivity Analysis

## Purpose

The main manuscript (`Pan_et_al_Revised_AIDS.qmd`) excludes individual-level socioeconomic variables (income, education, insurance) from primary models to avoid over-adjustment of the Non-Hispanic Black (NHB) race/ethnicity coefficient. The theoretical basis is VanderWeele & Robinson (2014): insurance is on the causal pathway between structural racism (indexed by NHB race/ethnicity) and cardiovascular risk. Including it as a covariate in a non-mediation framework would absorb part of the race-indexed disparity the study aims to estimate.

However, the Limitations section acknowledges that the NHB coefficient may overestimate the direct contribution of race-indexed exposures due to residual confounding from unmeasured SES. This sensitivity analysis directly addresses that concern by re-running the three main regression models with insurance type added as a covariate, allowing readers to see how much — if at all — the NHB coefficient attenuates.

**Insurance status is NOT added to the main analysis models.** It remains sensitivity-only.

## Insurance Variable: `ins_3cat`

The dataset contains six insurance variables: `INSURANCE`, `PRIVATE`, `MCAID`, `MCARE`, `VAOTHER`, `OTHERHC`. A 3-category variable is preferred over a binary (any vs. none) because quality of access differs meaningfully across insurance types: Medicaid enrollees face access barriers that privately insured patients do not, even though both technically have coverage.

| Category | Source | Label |
|---|---|---|
| Uninsured | None of the coverage flags = 1 | "Uninsured" (reference) |
| Public | `MCAID == 1` OR `MCARE == 1` OR `VAOTHER == 1` | "Public (Medicaid/Medicare/VA)" |
| Private | `PRIVATE == 1` | "Private" |

**Priority rule:** If both private and public flags are active, the participant is classified as Private (most favorable access). Uninsured is the reference category.

**Rationale for 3 categories over binary:** The NHB disparity framing in this paper concerns differential *quality* of healthcare access, not merely presence or absence of coverage. Medicaid and private insurance represent structurally different access levels, so collapsing them into "any insurance" would obscure this dimension.

## How to Run

Open `x24067_sensitivity_insurance.qmd` in RStudio and click **Render**, or run from the terminal:

```bash
quarto render "x24067_sensitivity_insurance.qmd"
```

The script reads `../data/prevent_risk.sas7bdat` (relative to its location in the `Sensitivity analysis/` folder).
