Supplementary analysis feasibility check

Sex-by-HIV supplementary table: feasible from the local prevent_risk.sas7bdat file and the existing analysis derivations.

Recommended variables for the dedicated sex-by-HIV table:
- HIV status: use STATUS with the existing QMD derivation hiv_status (STATUS == 1 -> PLWoH; STATUS in 2, 4, 5 -> PLWH). Do not use hiv2 unless reconfirmed against the current QMD.
- Sex: use SEX, matching the existing manuscript and model code. Do not use GENDERID for this table.
- Age: AGEATVIS.
- HAD outcomes: heart_age_deviation and derived had5 (heart_age_deviation >= 5).
- Race/ethnicity: use RACE and HISP with the existing QMD derivation race_eth_4cat.
- Region: REGION with the existing QMD derivation REGION_factor.
- Smoking: CURSMOKE_BIN.
- BMI/obesity: BMI and/or derived bmi_cat.
- Diabetes: DIAB_BIN or diabetes, consistent with the existing QMD derivation.
- Hypertension/SBP: SYSTOLIC and/or HYPERTENSIONMED, with sbp_cat if using the existing QMD category.
